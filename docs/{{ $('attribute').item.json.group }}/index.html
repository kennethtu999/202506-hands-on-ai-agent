台股最近一個交易日收盤行情

收盤行情

台股最近一個交易日（2025年5月29日）收盤：

  • 終場下跌10點，以21,347點作收。
  • 成交量擴張至3,322億元，呈現價跌量增格局。
  • 受端午連假前賣壓及MSCI半年度調整效應影響，盤中爆量甩尾翻黑。
類股表現
  • 電子股相對突出，但整體市場因賣壓影響震盪劇烈。
  • 防疫族群出現漲勢回升，表現較為強勢。
熱門個股
  • 永裕(1323)：將於6月5日除息，每股配發現金股利0.9元，預估參考價19.9元。
  • 台積電ADR：站回所有均線，有利台積電後市表現，成為市場關注焦點之一。
  • 輝達相關概念股：受輝達展望正向、美國四大雲端服務供應商資本支出強勁及沙烏地阿拉伯擬大舉投資AI等利多消息推動，相關科技類股人氣上升。
強勢漲停股

目前資料未明確列出當天具體的強勢漲停個股名單，但從防疫族群和部分電子權值股的活躍度來看，這些板塊中可能存在漲停或接近漲停的股票。投資人可關注相關產業龍頭及政策扶持標的。

總結與展望

雖然台股當天因賣壓與調整因素小幅下挫，但外資在五月份累計大幅買超逾2,199億元，是近18個月最大規模，有助於後續市場反彈動能。

展望六月，在美國科技巨頭及AI投資熱潮帶動下，以及台積電等權值電子股基本面改善下，台股仍有機會挑戰年線並持續反彈。
